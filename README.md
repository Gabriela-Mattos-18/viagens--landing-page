# viagens--landing-page
Projeto Devstart

Hidden Villages Europe — Landing Page

Este projeto é uma landing page criada para apresentar vilarejos pouco conhecidos da Europa. A página foi desenvolvida com HTML e CSS, seguindo requisitos de estrutura semântica, navegação interna e um banner principal com texto sobreposto.

Sobre o projeto

A ideia central é destacar destinos tranquilos e autênticos, oferecendo um conteúdo que valoriza a cultura local e a experiência de viagem em vilas europeias escondidas. O site apresenta informações de forma organizada e visualmente agradável.

Estrutura da página
Home (Banner)

Apresenta uma imagem representando um vilarejo europeu, com um texto sobreposto que introduz o tema do site.

MeetUs

Explica quem somos e o propósito do projeto, mostrando o compromisso em apresentar destinos verdadeiros e pouco explorados.

Advice

Reúne orientações para quem deseja viajar para vilas escondidas, destacando a importância da calma, do respeito local e da abertura para novas experiências.

Como executar o projeto

Baixe ou clone o repositório:

https://github.com/seu-usuario/viagens--landing-page


Abra o arquivo index.html no navegador.

Caso use Visual Studio Code, é possível carregar a página usando a extensão Live Server.

Também é possível publicar no GitHub Pages usando um link como:

https://seu-usuario.github.io/viagens--landing-page/
